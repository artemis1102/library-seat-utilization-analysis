# Library Seat Utilization Analysis: Understanding Perceived Seat Shortages Through Occupancy Data

## Background

Students often perceive that library seats are unavailable despite visible empty seats.

## Objective

Identify spatial and temporal factors affecting seat utilization.

## Dataset

- 478 active seats
- 665,156 occupancy observations
- Survey responses

## Methodology

- Data preprocessing
- Chi-square test
- Gini coefficient
- Correlation analysis
- Heatmap visualization

## Key Findings

1. Seat utilization is highly uneven.
2. Entrance proximity increases occupancy.
3. Occupancy peaks during exam periods.

## Policy Suggestions

- Reduce auto-return time
- Improve unpopular seat environments
- Detect fake occupancy behavior
